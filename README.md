# 🎨 SketchPad Pro

A modern and feature-rich drawing application built with **Python**, **Tkinter**, and **Pillow**. SketchPad Pro provides an intuitive interface for creating digital sketches, drawings, and simple artwork. It includes multiple drawing tools, customizable brushes, image editing capabilities, theme switching, and optional AI-powered sketch analysis.

---

## ✨ Features

### 🖌 Drawing Tools
- Freehand Pen
- Marker Tool
- Eraser
- Straight Line Tool
- Rectangle Tool
- Circle Tool
- Fill Bucket
- Text Tool
- Optional Grid Overlay

### 🎨 Customization
- Adjustable Brush Size
- Adjustable Brush Opacity
- Color Picker
- Built-in Color Palette
- Canvas Background Color Changer
- Live Brush Preview

### 🖼 Image Editing
- Open Existing Images
- Edit Images
- Create New Canvas
- Undo Support
- Export Drawings

### 💾 Export Formats
- PNG
- JPG
- PDF

### 🌙 User Interface
- Dark Theme
- Light Theme
- Responsive Toolbar
- Status Bar
- Mouse Coordinate Display
- Keyboard Shortcuts

### 🤖 AI Sketch Recognition *(Optional)*
- Analyze sketches using Claude AI
- Object Recognition
- Drawing Description
- Confidence Estimation
- Creative Improvement Suggestions

# 📦 Installation

## 1. Clone Repository

```bash
git clone https://github.com/yourusername/sketchpad-pro.git
```

---

## 2. Navigate to Project

```bash
cd sketchpad-pro
```

---

## 3. Install Dependencies

```bash
pip install Pillow anthropic
```

or

```bash
pip install -r requirements.txt
```

---

# ▶ Run the Application

```bash
python main.py
```

# 🤖 AI Sketch Analysis

The application supports optional AI-powered sketch analysis using the Anthropic Claude API.

### Capabilities

- Detect drawn objects
- Describe sketches
- Estimate confidence level
- Suggest improvements

> Install the `anthropic` package and configure your API key before using this feature.
